# proyecto-semestral-desarrollo-fs
Sistema de gestión y administración orientado principalmente a cafeterías.


## Requerimiento para ejecutar con PostgreSQL

En la clase main de cada microservicio, dar click derecho en la flecha verde a la izquierda de la clase,
click en "modify run configurations", click en "modify options" y marcar casilla "Add VM options", 
en esta escribir el siguiente comando "-Duser.timezone=UTC"
